善用你的搜索引擎

一、善用谷歌搜索

谷歌是目前最强大的搜索引擎，但是我们得会合理运用才能搜索出想要的内容，以下列出了一些谷歌搜索常用的高级表达式。当然，不仅仅适用于谷歌，三大搜索引擎都适用

1.精准搜索

  • 用法： 给关键词加上双引号
  • 效果： 强制搜索引擎匹配完整的短语，不拆分分词
  • 实战： 寻找报错信息、特定代码片段，或者通过“番号/ID”找资源

2.搜索对应格式文件

  • 注意： 冒号后面不要有空格
  • 实战： 机器学习入门 filetype:pdf 或 财务报表 filetype:xlsx。直接定位专业文档，跳过无意义的网页

3.站内搜索关键词

  • 用途： 当某个站点的原生搜索很难用时（比如某些政府官网或老旧论坛），直接调用谷歌的索引
  • 实战： python 多线程 site:zhihu.com（只看知乎里的讨论）

4.必须含有(或排除)指定内容

  • 排除（-）： 减号前有空格，后无空格。极其好用，如 Java教程 -site:csdn.net（屏蔽某站点的内容）
  • 包含（””）： 现在的谷歌中，+ 号已基本失效，强制包含请直接用双引号

5.逻辑操作符（OR）

  • 注意： OR 必须大写
  • 实战： Ubuntu OR Debian 安装教程。当你对某个名词有两种叫法不确定时使用

6.锁定搜索范围

常见用途：一定程度上防止内容农场的干扰

搜索标题为关键字的网站

7. 通配符与数字范围

  • * 通配符： 模糊匹配。如 如何解决 * 报错。
  • .. 数字范围： 搜索价格或年份。如 显卡 3000..5000 元 或 2022..2024 最佳论文

二、善用GitHub搜索

GitHub 提供了便捷的搜索方式，可以限定只搜索仓库的标题、编程语言、描述、README等。

1、根据仓库名称、描述、readme查找

查找仓库名称包含“关键词”的仓库

查找描述含有“关键词”的内容

查找README文件含有“关键词”的内容

2、根据star或fork数查找

表示查找star数或fork数大于“数字”且含有“关键字”的项目

  • 实战： 找质量靠谱的项目。如 stars:>5000 爬虫，帮你瞬间过滤掉个人练习作品。

3、根据仓库更新时间查找

表示在“日期”之后还在更新的且包含“关键字”的项目
其中“日期”的格式为年-月-日，如2023-04-24

4、根据仓库的编程语言查找

表示主要编程语言为“编程语言”的且包含“关键字”的项目

5、根据仓库的license查找

表示协议为“协议名”且含有关键字的仓库

6、根据仓库所有者查找

表示仓库所有者为“名称”的且含有“关键字”的仓库
